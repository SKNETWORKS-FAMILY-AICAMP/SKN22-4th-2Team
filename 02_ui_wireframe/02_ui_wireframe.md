## 1. 로그인 / 회원가입 (오버레이)

```
┌──────────────────────────────────────┐
│   [  로그인  ] | [  회원가입  ]       │  ← 탭 전환
├──────────────────────────────────────┤
│  이메일    [ user@example.com      ] │
│  비밀번호  [ ••••••••          👁  ] │
│  (회원가입만) 비번 확인 + 강도 바     │
│                                      │
│         [  🔑 로그인 / 가입  ]       │
│  ⚠️ 에러 메시지 (실패 시 표시)        │
└──────────────────────────────────────┘
```

| 상태    | 처리                     |
| ------- | ------------------------ |
| 제출 중 | 버튼 비활성화 + 로딩     |
| 성공    | 오버레이 닫힘, 헤더 갱신 |
| 실패    | 에러 토스트 표시         |

## 5. 에러 화면

```
┌──────────────────────────────────────┐
│           (이모지)                   │
│         에러 제목                    │
│       에러 설명 메시지               │
│  [ 다시 시도 ]   [ 새 아이디어 ]     │
└──────────────────────────────────────┘
```

**에러 유형별 메시지**

| 코드                | 이모지 | 제목                        | 버튼                       |
| ------------------- | ------ | --------------------------- | -------------------------- |
| `RATE_LIMITED`    | 🚦     | 분석 한도에 도달했습니다    | 내일 다시 / 검색 기록 보기 |
| `SESSION_EXPIRED` | 🔄     | 세션이 만료되었습니다       | 다시 시도                  |
| `TIMEOUT`         | ⏱️   | 분석 시간이 초과되었습니다  | 다시 시도 / 새 아이디어    |
| `NOT_FOUND`       | 📭     | 유사 특허를 찾지 못했습니다 | 새 아이디어 입력           |
| `NETWORK_ERROR`   | 🔌     | 서버 연결에 실패했습니다    | 다시 시도 / 새 아이디어    |

## 📐 공통 규칙

### 반응형

| 화면 크기            | 레이아웃                 |
| -------------------- | ------------------------ |
| 모바일 `< 768px`   | 사이드바 숨김, 단일 컬럼 |
| 태블릿 `768px~`    | 2열 (사이드바 + 메인)    |
| 데스크탑 `1280px~` | `max-w-7xl` 중앙 정렬  |

### 사용자 플로우

```
[진입] → [로그인] → [아이디어 입력] → [분석 시작]
  → [로딩] → [결과 확인] → [PDF 저장 or 재분석]
               └─ 에러 발생 시 → [에러 화면] → [다시 시도]
```
